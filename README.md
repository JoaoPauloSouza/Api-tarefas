# API de Tarefas

Este projeto é uma API simples de gerenciamento de tarefas (To-Do), desenvolvida com Python utilizando FastAPI e banco de dados SQLite.

A API permite criar tarefas, listar tarefas, marcar como concluídas e deletar tarefas.

## Tecnologias utilizadas

- Python
- FastAPI
- SQLite
- Docker

## Como rodar o projeto

### Rodando com Docker

No terminal, dentro da pasta do projeto, execute:

```bash
docker compose up --build
